Struktur Folder dan File Luzuno

/luzuno
│── index.html       → Halaman utama 
│── login.html       → Halaman login
│── register.html    → Halaman daftar
│── forum.html       → Halaman forum 
│── materi.html      → Halaman materi
│── style.css        → Gaya tampilan (CSS)
│── script.js        → Logika interaktif (JavaScript)
└── assets/
    ├── images/
    │   ├── logo.png     → Logo Luzuno
    │   ├── bg.jpg       → Background hero section
    │   └── icon.svg     → Ikon tambahan
    ├── fonts/           → Font khusus (jika ada)
    └── js/              → Script tambahan (jika diperlukan)
